    Welcome to StoneWare - A project made by Veteraning to try and find new ways of Embedding.
    This was EXTREMELY inspired by SandStone and buypass way of making a "pr0xy" in a single
    HTML file. This is very different tho... SandStone way of bypassing extension based web
    filters way by rewriting the URL, iframing it with a sandbox and a lot of other stuff while
    this one works by using nieche and unknown ways to embed sites like the object tag and others
    it also uses cors proxies that are used to scrape to instead load pages. This is mainly built
    for playing online games like 1v1.lol and Run3 not to use as an actual proxy or run more
    robust online games lik Eaglercraft and other online ports like Hollow Knight and others.
